# Programming Learning Guide

初心者向けのプログラミング学習教材をまとめたリポジトリです。

## 目的
プログラミング初学者が、基礎文法や考え方を理解しやすいように、
サンプルコードと簡単な解説をセットで整理しています。

## 対象技術
- Java
- Python
- SQL

## 想定読者
- プログラミング初学者
- 文法の基礎を復習したい人
- 学習用サンプルを探している人

## 構成
- `java-basics/` : Javaの基礎
- `python-basics/` : Pythonの基礎
- `sql-basics/` : SQLの基礎
