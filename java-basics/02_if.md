# if文

if文を使うと、条件によって処理を分けることができます。

## サンプルコード

```java
public class IfSample {
    public static void main(String[] args) {
        int score = 80;

        if (score >= 60) {
            System.out.println("合格");
        } else {
            System.out.println("不合格");
        }
    }
}
